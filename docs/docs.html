CLI commands

zshield start

Start the DNS resolver daemon. Listens on localhost and intercepts DNS queries, blocking domains that match configured blocklists.

$ zshield start

zshield

Attach the interactive TUI dashboard. Shows real-time query log, blocking stats, and quick access to allow/deny overrides.

$ zshield

zshield status

Print current blocking statistics — total queries, blocked count, top blocked domains.

$ zshield status

zshield allow <domain>

Whitelist a domain. Queries for this domain will always resolve normally, even if it appears on a blocklist.

$ zshield allow example.com

zshield block <domain>

Blacklist a domain. Queries for this domain will always be blocked, regardless of blocklists.

$ zshield block tracker.example.com
